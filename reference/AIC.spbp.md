# Akaike information criterion for fitted spbp models

Akaike information criterion for fitted spbp models

## Usage

``` r
# S3 method for class 'spbp'
AIC(object, ..., k = 2)
```

## Arguments

- object:

  A fitted `"spbp"` object.

- ...:

  Additional fitted models for comparison.

- k:

  Penalty per parameter (default 2).
