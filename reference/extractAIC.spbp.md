# Extract AIC from a fitted spbp model

Extract AIC from a fitted spbp model

## Usage

``` r
# S3 method for class 'spbp'
extractAIC(fit, scale = 0, k = 2, ...)
```

## Arguments

- fit:

  A fitted `"spbp"` object.

- scale:

  Not used.

- k:

  Penalty per parameter (default 2).

- ...:

  Not used.
