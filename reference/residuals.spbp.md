# BP based models residuals.

Residuals for a fitted
[`spbp`](https://rvpanaro.github.io/spsurv/reference/spbp.md) model.

## Usage

``` r
# S3 method for class 'spbp'
residuals(
  object,
  type = c("martingale", "deviance", "cox-snell", "coxsnell"),
  ...
)
```

## Arguments

- object:

  an object of class \`spbp\` result of a
  [`spbp`](https://rvpanaro.github.io/spsurv/reference/spbp.md) fit.

- type:

  type of residuals, default is "cox-snell"

- ...:

  arguments passed to parent method.

## See also

[`spbp`](https://rvpanaro.github.io/spsurv/reference/spbp.md),
[`spbp`](https://rdrr.io/pkg/survival/man/survfit.html).
