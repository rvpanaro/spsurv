# Standard errors for fitted spbp model parameters

Standard errors for fitted spbp model parameters

## Usage

``` r
se(object, ...)
```

## Arguments

- object:

  A fitted `"spbp"` object.

- ...:

  Not used.

## Value

Named numeric vector of standard errors aligned with
[`estimates`](https://rvpanaro.github.io/spsurv/reference/estimates.md).
